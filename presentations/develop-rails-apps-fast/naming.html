Naming conventions

Suffix returned object type 

  
  event.cohost
  # => CompanyUser #123....

  journey.last_edited_by
  # => CompanyUser #123....

Suffix returned object type 

  
  event.cohost_company_user
  # => CompanyUser #123....

  journey.last_edited_by_company_user
  # => CompanyUser #123....

Don't be shy of long names 

  
  # IDs of +FeedbackCycleFeedbackTypes+
  #     which are +blocked+
  #     for the +reviewee+
  feedback_cycle_feedback_types_ids_with_blocked_access_for_reviewee

Predicate in the variable name 

  
  # simple variables
  has_admin_rights = @company_user.admin?

  # conditioning arguments
  Slack::ChannelsList.fetch_all!(
    should_include_private_channels: true
  )
Modules are Plural
Classes are Singular 
  
  Companies::CompanyUsersList
    .new(companys)
    .all

  Events::SessionsList
    .new(events)
    .all
